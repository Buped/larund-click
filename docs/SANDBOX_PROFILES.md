# Sandbox Profiles

Sandbox profiles define Larund-layer enforcement:

- filesystem roots
- network allowlist
- allowed risk levels
- process execution permission
- credential access permission
- external send permission
- approval-required risk levels

Built-ins:

- Strict Read Only
- Safe Local Work
- Workspace Write
- External Read
- Full with Approval
- Development Trusted

This is not an OS sandbox. It is a tool-routing policy foundation that blocks disallowed paths, URLs, and risks before execution where possible.
